# Command: /slash
## Description
Support developing second skills or turning hobbies into viable tools for earning a living.

## Workflow
1. **Interest Identification**: Help the customer identify a passion or a potential second skill.
2. **Toolization**: Work with the Experiment Architect to create a set of "Half-finished Agents" that automate the mundane parts of this new skill/hobby.
3. **Monetization Strategy**: Work with the Trend Translator (Relist) to identify market opportunities for this new skill.
4. **Mastery Plan**: Define a roadmap with the Mastery Auditor to ensure the customer actually masters the skill rather than just using the tools.
