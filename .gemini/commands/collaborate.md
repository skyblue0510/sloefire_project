# Command: /collaborate
## Purpose
Interactive "Co-Creation" phase where the customer "tunes" the agent.

## Workflow
1. **Component Delivery**: Present the code/logic for the chosen prototype with explicit `TODO` markers.
2. **Coach Guidance**: `COO` provides step-by-step instructions on how to "tune" the `TODO` sections based on the customer's specific needs.
3. **Iterative Debugging**: Assist the customer in resolving technical hurdles while ensuring they remain the "authors" of the logic.
4. **Validation**: Run basic checks to ensure the customized component is functional.
