# Command: /optimize
## Purpose
Analyze the co-creation process and update the mastery profile.

## Workflow
1. **Behavioral Analysis**: `Mastery Auditor` scans the session logs for signs of "legacy thinking" or "increased mastery."
2. **Mastery Update**: Update `shared-memory/client/current/mastery_report.md` with new insights and "negative feedback."
3. **Refinement**: Suggest tweaks to the current agent components based on the mastery level.
4. **Output**: Present the progress report to the customer.
