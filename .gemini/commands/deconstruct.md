# Command: /deconstruct
## Purpose
Move from strategic theme to technical prototypes.

## Workflow
1. **Hypothesis Generation**: Invoke `Experiment Architect` to propose 2-3 "Experiment Hypotheses" based on the approved theme.
2. **Prototype Blueprint**: For each hypothesis, outline the "Half-finished Agent" components (e.g., specific Python modules, Prompt chains).
3. **Logic Preview**: Explain the "non-normative" logic behind these components.
4. **Output**: Present the hypotheses and prototypes to the customer, asking which one they want to start "co-creating."
